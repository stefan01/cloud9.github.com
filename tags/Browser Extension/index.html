Yawning Search Bar Demo 23 Nov 2010



UI is essentially the a element of the web, and I wanted to explore the affordances of it with playfulness in mind.

Why webpages should look the same? Can web space be less virtual by preserving the nuances and qualities that we treasure in the physical world? This Chrome Extension experiment is re-imaging the traditional UI elements in the browser, instead of having the generic looks, UI are shaped with freehand drawing strokes and the search bar is demonstrating the yawing behavior when the user opens the google page.

This experiment is a Chrome Extension which I programmed in javascript and processing.js as drawing library.

Web Graffiti for Self-Expression 22 Nov 2010

Can virtual space provide corners and walls for people to comment, to create, to shout, to be heard, to play, and to share, but in other forms of blogging or twitter feeds?

I think browser extension and html5 canvas is good platform to start, for it’s ability to easily integrated with existing browser, and access to the web.

UI is a significant element of webpages, and with the html5 Canvas, essentially we can draw on the it.
